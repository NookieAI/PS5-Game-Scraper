# PS5 Game Scraper

A Windows, Linux and Mac desktop application built with Electron to scrape and display PS5 game download links from dlpsgame.com. It extracts links from popular hosts like Akira, Viking, and 1Fichier, along with game details such as firmware compatibility, descriptions, and screenshots.

## Features

- **Fast Scraping**: Efficiently fetches game lists from category pages and RSS feeds for quick results.
- **Download Links**: Extracts and organizes download links by host and version (e.g., v1.00, v1.02).
- **Firmware Information**: Automatically detects and displays the lowest supported firmware version (e.g., "2.xx and higher") for each game in the modal view.
- **Game Details**: Includes voice/subtitle info, notes, size, descriptions, and up to 2 screenshots per game.
- **Favorites**: Mark games as favorites for easy access.
- **Sorting & Filtering**: Sort by date or name; filter by favorites; search games by title.
- **Modal View**: Click on any game card for detailed info, links, and images.
- **Caching**: Saves data locally for faster loading on subsequent runs.
- **Cross-Platform**: Works on Windows, macOS, and Linux via Electron.
