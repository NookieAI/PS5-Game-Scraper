# PS5 Game Scraper

A sleek and powerful desktop application built with Electron that scrapes and displays PS5 game information from dlpsgame.com. Designed for gamers who want an easy way to discover, search, and download PS5 games with Akira and Viking links.

## Key Features

- **Automated Game Scraping**: Fetches the latest PS5 games, including high-quality covers, release dates, voice/subtitle info, and more.
- **Advanced Search & Filtering**: Real-time search by game title. Toggle to view only your favorite games.
- **Flexible Sorting**: Sort games by release date or alphabetically by name.
- **Favorites System**: Star games to mark them as favorites for quick access.
- **Detailed Game Modals**: Click any game card to view comprehensive details:
  - Game cover and metadata (voice, subtitles, notes, size).
  - Formatted descriptions with headings and bullet points.
  - Up to 2 screenshots per game.
  - Direct download links for Akira and Viking versions.
- **Full-Size Image Viewer**: Click on screenshot thumbnails to view full-size images inside a dedicated modal.
- **Progress Tracking**: Animated progress bar and text updates during scraping operations.
- **Caching & Performance**: Stores data locally to avoid repeated downloads. Clear cache anytime.
- **Notifications**: Desktop notifications for scan completion or cancellation.
- **Modern UI**: Responsive design with glassy, transparent buttons for a polished look. Adapts to various screen sizes.
- **Cross-Platform**: Runs on Windows, macOS, and Linux via Electron.
